Bước 1: Tạo dữ liệu & Logic lật thẻ (Flashcard Logic)
    Hiện tại app chỉ có 1 thẻ "Vocabulary" chết cứng.
    Cần: Tạo một danh sách từ (Array) trong Javascript -> Hiển thị từng từ lên -> Bấm để lật xem nghĩa -> Bấm "Next" để qua từ mới.
Bước 2: Tính năng "Thêm từ mới"
    Tạo một nút "+" và một Form để bạn tự điền từ vựng muốn học vào.
Bước 3: Lưu trữ dữ liệu (Local Storage)
    Hiện tại nếu F5 (tải lại trang) là mất hết. Cần lưu xuống bộ nhớ trình duyệt để lần sau mở app lên vẫn còn.

#
Nếu muốn reset lại dữ liệu mẫu ban đầu, bạn có thể xóa cache trình duyệt hoặc dùng lệnh console localStorage.clear()

#
Bấm tổ hợp phím Ctrl + F5 (hoặc Cmd + Shift + R trên Mac) để xóa cache và tải lại trang.
